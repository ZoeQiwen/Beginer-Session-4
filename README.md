# 🎲 Random Module Practice Project

This is a simple Python project to practice using the `random` module. It demonstrates how to generate random numbers and use them in decision-making, similar to a coin flip program.

---

## 📌 Features

* 🎯 **Random Number Generator**
  Generates a random number between 0 and 1.

* 🪙 **Simple Decision (Coin Flip)**
  Outputs different results based on the random number (like heads or tails).

---

## 🧠 Concepts Used

* `random` module
* `randint()` function
* Conditional statements (`if`)
* Basic program logic

---
